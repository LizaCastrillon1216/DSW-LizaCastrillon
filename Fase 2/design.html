Sitio web para el aprendizaje de la Interfaz de usuario e interacción Humano ordenador

Este sitio web tiene como proposito servir como objeto virtual de aprendizaje sobre las tematicas de interfaz de usuario y la interaccion Humano servidor.

Interfaz de usuario

Una interfaz de usuario es el conjuntos de controles y canales sensoriales mediante los cuales un usuario puede comunicarse con una máquina.

Workana LLC . (2020). Workana. Obtenido de ¿Qué es Interfaz de usuario?: https://i.workana.com/glosario/interfaz-de-usuario/

  • Tipos de interfaz de usuario:
    • Interfaz de voz: Son programas capaces de identificar comando de voz.
    • Interfaz de texto: Se utiliza principalmente en la programación.
    • Interfaz natural: Es la interfaz que capaz de identificar e interpretar acciones naturales.
    • Interfaz cerebro-ordenador: Esta es la mas innovadora hasta el momento pero se pretende controlar prótesis biónicas y dar instrucciones sencillas
      a un software por medio de hondas cerebrales.
    • Interfaz gráfica: Es el contenido gráfico por medio del cual se puede visualizar la información del equipo en la pantalla.
    • Interfaz de sitios web: son los gráficos, información y herramientas que aparecen en la pantalla al abrir una página web.

Interacción Humano-Ordenador.

La interacción entre el humano y el computador se puede definir tambien como la comunicación existente entre un emisor y un receptor, pero este proceso no
solo comunica un mensaje, tambien permite emitir un estímulo el cual desarrolle una respuesta en tiempo real. En la actiualidad el intenet actúa como un
organismo social que permite una relación interactiva entre el usuario y la red, debido a que al momento en que la red genera un estímulo a través de la
página web, solicitada por el usuario, el usuario decide con cual interfaz desea establecer un dialogo comunicativo.
En esta interacción el diselo de la interfaz gráfica de usuario cumple un rol increiblemente significativo en la mediación de las distintas relaciones
cognitivas que establece el usuario cuando interactua con el ordenador. La comprensión del diseño y desarrollo de las interfaces hace mas posible o imposible
la comunicación entre el usuario y el ordenador.

Jordán, E. d., Muñoz, V. I., & Aparicio, D. M. (03 de Abril de 2020). RECIAMUC. Obtenido de Interacción humano - tecnología, interfaces y usabilidad: https://reciamuc.com/index.php/RECIAMUC/article/view/475/737
La interacción Humano-Ordenador no solo depende de la interfaz, sino que también depende de la usabilidad ya que los principios de la misma son la base de
cualquier página web para que sea "User Friendly".
